# Sound Files

Add the following sound files to this directory:

## Required Sounds

| Filename | Description | Suggested Duration |
|----------|-------------|-------------------|
| `click.mp3` | UI button click | 0.1s |
| `correct.mp3` | Correct action/answer | 0.5s |
| `wrong.mp3` | Wrong action/answer | 0.3s |
| `complete.mp3` | Level complete celebration | 1-2s |
| `hint.mp3` | Hint reveal | 0.3s |
| `drop.mp3` | Item dropped | 0.2s |
| `pickup.mp3` | Item picked up | 0.2s |
| `swoosh.mp3` | Swipe/throw sound | 0.3s |
| `pop.mp3` | Item appears/disappears | 0.2s |
| `star.mp3` | Star earned | 0.4s |
| `button.mp3` | Menu button press | 0.1s |
| `background-music.mp3` | Background loop | 60-120s (loop) |

## Recommended Sources (Free)

1. **Freesound.org** - Free sound effects (requires attribution)
2. **Mixkit.co** - Free sound effects
3. **Pixabay.com/sound-effects** - Free sound effects
4. **OpenGameArt.org** - Free game assets

## Audio Format

- Format: MP3 (best compatibility)
- Sample Rate: 44.1kHz
- Bit Rate: 128-192 kbps
- Channels: Stereo or Mono

## Tips

- Keep files small for faster loading
- Normalize audio levels for consistency
- Test on mobile devices for volume balance
